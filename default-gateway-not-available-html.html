How to Fix The default gateway is not available on Windows 10

People are currently reading this guide.

What is Default Gateway on Windows networking?

The default gateway is the node of your system that transfers packets to external networks from your network. Or you can say the default gateway is a point of access or an IP router used by a networked computer to send data to a computer in another network or on the web.

Why is the default gateway not available?

Because of the wrong IP settings, your default gateway may not be available. This may even be caused in rare instances by the setup of the ISP.

This issue could also be caused by router setups and outdated drivers. Some users noted that safety solutions from McAffee may sometimes block your link.

Solution 1: Uninstall Network Adapters Driver

1. To open Device Manager, press Windows Key + R and type "devmgmt.msc."

2. Expand Network adapters and search for the name of your network adapter.
3. Ensure that you notice the name of the adapter if something goes incorrect.
4. Right-click and uninstall your network adapter.


5. Select Yes if you ask for confirmation.
6. Restart your computer and start trying to reconnect to your network.
7. If the network can not be connected, then driver software is not installed automatically.
8. Now you must visit the website of your manufacturer and download the driver.
9. Install the driver and restart the PC.

Solution 2: Update Network Adapters Driver

1. To open the device manager, press the Windows button + R and type "devmgmt.msc."
2. Expand Network Adapter then right-click on your WiFi Controller and choose Update Driver Software (e.g. Broadcom or Intel).



3. Choose "Browse my computer for driver software" from the Update Driver Software Windows.


4. Now select “Let me pick from a list of device drivers on my computer.“


5. Uncheck now Show hardware-compatible option.
6. Select Broadcom from the left menu from the list, and then select the Network Adapter Broadcom 802.11a on the correct tab.


7. Finally, if the confirmation is requested, press Yes.
